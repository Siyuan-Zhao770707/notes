#### 实现简单的服务通信

**public**:
	1. 创建服务器端的对象
	2. 通过回调函数处进行服务
	3. 向客户端反馈应答结果
注释：这里使用了自定义的通信接口AddTwoInts，具体是怎么样定义和使用可见[[通信接口]]
```python
import rclpy
from rclpy.node import Node
from learning_interface.srv import AddTwoInts


class adderServer(Node):

    def __init__(self, name):
        super().__init__(name)
        # 创建服务端（接口类型，服务名，回调函数）
        self.srv = self.create_service(AddTwoInts, 'add_two_ints', self.adder_callback)

    def adder_callback(self, request, response):
        """收到客户端请求，计算两数之和并返回"""
        response.sum = request.a + request.b
        self.get_logger().info('Incoming request\na: %d b: %d' % (request.a, request.b))
        return response  # ROS2 自动将 response 发回客户端


def main(args=None):
    rclpy.init(args=args)
    node = adderServer("service_adder_server")
    rclpy.spin(node)    # 阻塞等待客户端请求
    node.destroy_node()
    rclpy.shutdown()
```

**client**:
	1. 创建客户端对象
	2. 创建并且发送请求数据 
	3. 等待服务器端的应答数据
