###  实现一个简单的hello world节点

**helloworld节点文件**
*所有节点文件的大致流程：
1. 程序接口初始化
2. 创建节点并且初始化
3. 实现节点功能
4. 销毁节点并且关闭窗口
#### 方法一：面向过程编程（适合简单任务）
*learning_node -> learning_node -> node_helloworld.py*:
```python
import rclpy #引入ros2的python库
from rclpy.node import Node #引入节点类 
import time #引进time模块

def main(args = None): 
	rclpy.init(args = args) #ros2的python接口初始化
	node = Node("node_helloworld") #创建节点类对象并且初始化，创建名字
	
	while rclpy.ok(): #ros2系统是否正常运行 
		node.get_logger().info("Hello World") #ros2日志输出（在终端输出字符串）
		time.sleep(0.5) #休眠0.5秒
		
	#结束程序
	node.destroy_node() #销毁节点对象
	rclpy.shutdown() #关闭ROS2的Python接口
```
#### 方法二：面向对象编程（适合复杂任务）
*learning_node -> learning_node -> node_helloworld_class.py*
```python
import rclpy
from rclpy.node import Node
import time

class HelloWorldNode (Node):
	def __init__(self, name):
		# 我们自己创建的类继承自ros2自带节点的父类
	 	super().__init__(name)# 初始化父类 
	 	# 我们自己定义的类功能
		while rclpy.ok():
			self.get_logger().info("Hello World")
			time.sleep(0.5)
	
def main(args=None):
	rclpy.init(args=args)
	node = HelloWorldNode("node_helloworld_class")#使用自己创建的类来进行初始化
	node.destroy_node()
	rclpy.shutdown()
```
