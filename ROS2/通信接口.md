#### 查看具体的数据结构

**方式一**： 到 *opt/ros/你的ros2版本/share/* 这个目录下面搜索你想要的数据类型
**方式二**：使用过命令行查看
```zsh
# 当前系统里面所有的接口的定义都列了出来
ros2 interface list 
# 参看具体的定义结构
ros2 interface show <接口定义名称>
# 某个功能包里面定义什么接口
ros2 interface package <功能包名称> 
```

####  定义接口（以.msg为例子）

**定义接口一定使用的是ament_C格式创建的功能包！**
**ros2要求接口名称的首字母必须大写！！**
 *learning_interface（包名称）-> serv（对应的通信类型） -> GetPosition.srv*（首字母大写）
 ---之上是回调函数的request部分，---之下是回调函数的response部分
 ```markdown
 bool get # 获取目标位置的指令 
 ---
 int32 x  # 目标的x坐标
 int32 y  # 目标的y坐标
 ```
 *learning_interface（包名称）-> CMakeList.txt
 结构可以参考[[工作空间与功能包#功能包的结构：]]
 在此处添加接口文件*
 注释：有两个地方的代码需要添加
 ```python
 find_package(rosidl_default_generators REQUIRED)
 rosidl_generate_interfaces(${PROJECT_NAME}
 "文件的相对路径")
 ```
 ![[截屏2026-07-13 23.12.26.png|685]]
learning_interface -> package.xml
```xml
<?xml version="1.0"?>
<?xml-model href="http://download.ros.org/schema/package_format3.xsd" schematypens="http://www.w3.org/2001/XMLSchema"?>
<package format="3">
  <name>learning_interface</name>
  <version>0.0.0</version>
  <description>TODO: Package description</description>
  <maintainer email="2285518665@qq.com">zhaosiyuan</maintainer>
  <license>TODO: License declaration</license>

  <buildtool_depend>ament_cmake</buildtool_depend>

  <test_depend>ament_lint_auto</test_depend>
  <test_depend>ament_lint_common</test_depend>
  
  <!-- 一定要加上这一行 -->
  <member_of_group>rosidl_interface_packages</member_of_group>

  <export>
    <build_type>ament_cmake</build_type>
  </export>
</package>

```

注意：如果你在定义通信接口的时候引用了其他的接口：
```markdown
geometry_msgs/Point center
float64 radius
```
那么你需要将被应用的接口也添加到CMakeList.txt和package.xml里面
```cmake
find_package(geometry_msgs REQUIRED)
find_package(rosidl_default_generators REQUIRED)

rosidl_generate_interfaces(${PROJECT_NAME}
  "msg/Num.msg"
  "msg/Sphere.msg"
  "srv/AddThreeInts.srv"
  DEPENDENCIES geometry_msgs # Add packages that above messages depend on, in this case geometry_msgs for Sphere.msg
)
```

```xml
<depend>geometry_msgs</depend>
<buildtool_depend>rosidl_default_generators</buildtool_depend>
<exec_depend>rosidl_default_runtime</exec_depend>
<member_of_group>rosidl_interface_packages</member_of_group>
```

#### 使用接口
python版本
```python
# from <包文件名>.<通信格式> import <接口名称>
from learning_interface.srv import GetPosition

```
C++版本
```cpp
//一定记住结尾是.hpp文件！！！！
#include "learning_interfaces/srv/GetPositon.hpp"
```
其他的通信类型也是类似
