 
**参数：**
	1. 机器系统的全局共享字典
	2. 字典由键和值组成
	3. 可以实现动态监控

#### 命令行操作参数
```zsh
# 查看当前系统有什么样的参数
ros2 param list
# 查看一个具体的参数的说明
ros2 param discribe <节点名称> <参数名称>
# 查看某个参数是具体数据
ros2 param get <节点名称> <参数名称>
# 将某个参数设置为某个具体的数
ros2 param set <节点名称> <参数名称>
# 在终端展示某个节点的所有参数
ros2 param dump <节点名称>
# 将某个节点的参数保存在一个文件里面，可以直接对参数文件进行修改
ros2 param dump <节点名称> >> <参数文件名称.yaml>
# 加载某个参数文件
ros2 param load <节点名称> <参数文件名陈.yaml>
```

#### 参数编程

```python
import rclpy
from rclpy.node import Node

class ParameterNode(Node):
    def __init__(self, name):
        super().__init__(name) 
        self.timer = self.create_timer(2, self.timer_callback)
        # 创建参数（键，值）
        self.declare_parameter('robot_name', 'mbot')

    def timer_callback(self):
		# 拿到参数值
		# get_parameter("键名称")定位键
		# get_parameter_value()定位键所对应的值
		# string_value这个值所对应的的数据结构（字符串）
        robot_name_param = self.get_parameter('robot_name').get_parameter_value().string_value

        self.get_logger().info('Hello %s!' % robot_name_param) 
        
        # 修改参数（实时初始化参数让其不变）
        # 将新参数的值赋值给一个临时变量
        new_name_param = rclpy.parameter.Parameter('robot_name',
                               rclpy.Parameter.Type.STRING, 'mbot')
        # 将参数值变成一个列表
        all_new_parameters = [new_name_param]
        # 将参数列表重新发给ros2系统
        self.set_parameters(all_new_parameters)

def main(args=None):
    rclpy.init(args=args)
    node = ParameterNode("param_declare")
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()
```
